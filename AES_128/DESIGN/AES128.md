# AES-128 Encryption Core – Project Documentation

## Project Title

**AES-128 Encryption Core Design and Verification using Verilog HDL**

## Software Used

* Xilinx Vivado 2023.2
* Verilog HDL

## Objective

To design and verify an AES-128 encryption core capable of encrypting 128-bit plaintext using a 128-bit secret key.

## AES Operations Implemented

* SubBytes
* ShiftRows
* MixColumns
* AddRoundKey
* Key Expansion

## Module Inputs

| Signal   | Width | Description        |
| -------- | ----- | ------------------ |
| clk      | 1     | System Clock       |
| reset_n  | 1     | Active-Low Reset   |
| IN_valid | 1     | Input Valid Signal |
| IN_state | 128   | Plaintext Input    |
| key      | 128   | Encryption Key     |

## Module Outputs

| Signal    | Width | Description         |
| --------- | ----- | ------------------- |
| OUT_valid | 1     | Output Valid Signal |
| OUT_state | 128   | Ciphertext Output   |
